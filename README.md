## Installation

##### Install Dependencies

``` 
yarn or npm install
```

##### Scripts

Run with `yarn <script>` or `npm run <script>`

| Command        | Description  |
| -------------  |-------------| 
| serve      | starts a dev server and opens a browser to port 3000  | 
| build   | builds app distribution in `/dist`     |
